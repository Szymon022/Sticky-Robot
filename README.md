# Obstacle Avoiding Robot - Sticky Robot

## Technologies used
C++, Arduino, Git

## What is Sticky Robot
Sticky Robot is an obstacle avoiding robot. The robot continues to move forward until it approaches an obstacle. It then makes a turn and continues to move forward.
